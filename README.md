# Hemanathan Portfolio

This is a static personal portfolio website ready to deploy for free with GitHub Pages.

## Files

- `index.html` - main portfolio website
- `Hemanth_Resume.pdf` - downloadable resume
- `.nojekyll` - tells GitHub Pages to serve the site as plain static files

## Publish Free With GitHub Pages

1. Create a new public repository on GitHub.
2. Upload all files from this folder to the repository.
3. Open the repository on GitHub.
4. Go to `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main`.
7. Select folder `/ (root)`.
8. Click `Save`.
9. Wait for GitHub to publish the site.

Your site URL will look like:

- `https://yourusername.github.io/repository-name/`

If the repository name is exactly `yourusername.github.io`, then the site URL becomes:

- `https://yourusername.github.io/`
